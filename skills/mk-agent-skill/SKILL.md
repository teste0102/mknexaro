# 🤖 MK Agent Skill

**Agente inteligente que executa qualquer tarefa**

## Como usar

```bash
/mk-agente seu comando aqui
```

## Exemplos

- `/mk-agente qual é o preço do Samsung A15?`
- `/mk-agente cria uma lista de compras`
- `/mk-agente me ajuda a organizar meu dia`

## O que faz

- ✅ Processa comandos em português
- ✅ Integra com todas as 13 skills
- ✅ Responde via Haiku 4.5
- ✅ Executa tarefas automaticamente
