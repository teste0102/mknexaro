PASSO 2: Coletar Dados Manualmente
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. Abra: https://shopee.com.br/search?keyword=tela+A15+4g

  2. Pressione F12 (DevTools)

  3. Vá para Console

  4. Cole este script:

     ▼ SCRIPT PARA COLETAR (cole no console) ▼
     ─────────────────────────────────────────
     const produtos = [];
     document.querySelectorAll('a[href*="/p/"]').forEach((link) => {
         const titulo = link.getAttribute('title') || link.innerText;
         if (titulo.length > 5) {
             produtos.push({
                 titulo: titulo.trim(),
                 preco: 'R$ ...',  // você preenche
                 desconto: '',
                 avaliacao: 'N/A',
                 vendas: 'N/A',
                 url: link.href,
                 fonte: 'Shopee',
                 pasta: 'shopee',
                 imagem: ''
             });
         }
     });
     copy(JSON.stringify(produtos, null, 2));
     console.log('✅ Copiado!');
     ─────────────────────────────────────────

  5. Pressione Enter

  6. Cole o resultado em um arquivo .json


PASSO 3: Abrir Excel e Preencher
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
  1. Abra o Excel criado no Passo 1

  2. Cole os dados que copiou

  3. Complete as colunas:
     - Título ✅ (do script)
     - Preço ✅ (complete manualmente)
     - Desconto (opcional)
     - Avaliação (você vê na página)
     - Vendas (você vê na página)
     - URL ✅ (do script)
     - Fonte: Shopee
     - Pasta: shopee
     - Imagem (opcional)

  4. Salve o arquivo

❓ PERGUNTAS FREQUENTES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

P: Por que não é automático?
R: Shopee tem proteção contra bots. Coleta manual é 100% confiável.

P: Posso coletar de outras fontes?
R: SIM! Crie pastas (mercado_livre/, olist/, etc.) e o comparador processa.

P: Quantos produtos posso coletar?
R: Ilimitado. Quanto mais scroll, mais produtos carregam na Shopee.

P: Os dados são salvos?
R: Sim! Tudo fica em ~/dados_produtos/ para análise futura.

P: Posso editar os dados depois?
R: SIM! Abra qualquer Excel e edite. Comparador usa a versão mais recente.
